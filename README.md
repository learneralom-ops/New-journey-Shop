# New journey Shop - E-commerce Website

A New journey Shop-style single page e-commerce website with Firebase integration and responsive design.

## Features

### User Side
- Single Page Application (SPA) design
- Modern Daraz-like UI with orange theme
- Product categories (Sex Toy, Electronics, Fashion, Grocery, Mobile, Accessories)
- Product listing with grid layout
- Product details modal
- Shopping cart with quantity control
- Checkout system with form validation
- User authentication (login/register)
- Search functionality
- Responsive design (mobile + desktop)

### Admin Panel
- Admin login system
- Dashboard with statistics
- Product management (CRUD)
- Category management
- Order management (approve/cancel)
- User management
- Responsive admin interface

## Tech Stack
- HTML5
- CSS3 (Flexbox, Grid)
- Vanilla JavaScript
- Firebase (Authentication, Realtime Database)
- Font Awesome icons

## Firebase Configuration

The project uses Firebase for:
- User authentication
- Product data storage
- Order management
- Real-time updates

## Setup Instructions

1. **Clone or download** the project files
2. **Open `index.html`** in a web browser to start the user side
3. **Open `admin.html`** to access the admin panel
4. **Admin credentials**:
   - Username: `admin`
   - Password: `admin123`
